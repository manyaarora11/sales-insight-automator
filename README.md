\# Sales Insight Automator



A containerized AI-ready system that converts raw CSV sales data into executive-level insights.



\## Features



\- CSV file upload API

\- Automated sales summary generation

\- FastAPI backend

\- Swagger API documentation

\- Docker containerization



\## Architecture



Frontend:

API tested via Swagger UI



Backend:

FastAPI service handling file upload and processing



AI Engine:

Prototype logic implemented for generating insights.

Can integrate with Gemini / Groq APIs.



\## Security



\- Input validation via FastAPI

\- Controlled file upload handling

\- Containerized runtime environment



\## Run Locally



Clone repository



docker compose up --build



Open Swagger:



http://localhost:8000/docs
